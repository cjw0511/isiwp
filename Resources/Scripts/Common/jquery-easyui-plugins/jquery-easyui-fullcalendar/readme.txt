插件说明：
1，概述：实现可支持全屏显示的日历组件，并且组件上的日期数字附加鼠标悬停事件，悬停时显示当日的附加信息例如农历、节日等。

2，引用方式：
    A，在需要使用该插件的页面引入 jeasyui 的相关基本库文件，例如 "easyui.css" 和 "jquery.easyui.min.js"；
    B，引入 "jquery.fullcalendar.js" 文件。

3，注意：
    A，请勿修改本文件夹下任何文件的名称，请勿删除本文件夹下任何文件；
